SCENARIOS

八大高频协同场景

面向销售、客户成功、HR、法务、教研、培训等核心岗位，每个场景含场景描述、适用对象、连接器与可直接复用的 Prompt 示例。
🎯
销售单场会议内容质检
01 / 08
销售拜访/电销场景下，通过腾讯会议录制谈话内容，会议结束后由 WorkBuddy 自动拉取智能纪要与转写文本，按企业自定义评分卡（开场/需求挖掘/方案讲解/异议处理/Close 五维）打分，输出可下载的销售行为评分报告与改进建议。
👥 适用对象销售、销售管理者、客户成功
🔌 连接器腾讯会议 + 乐享 + 企业微信
⚡ PROMPT 示例
帮我分析腾讯会议[会议号 123-456-789]的录制内容，按"开场破冰/需求挖掘/方案讲解/异议处理/Close 推进"五维评分（满分 100），结合转写原文给出每一维的扣分点和原话引用，最后输出评分报告，保存到乐享知识库"销售话术评分"文件夹，文件按"时间+录制文件名"命名，并把链接发给我。
📚
会议纪要自动归档到知识库
02 / 08
会议结束后，WorkBuddy 自动获取腾讯会议智能纪要 + 关键决议 + 待办，按业务类型（客户拜访/伙伴交流/内部评审）选择对应模板，写入乐享知识库对应空间，附上录制回放链接，并 @ 相关人员。
👥 适用对象销售、项目经理、各业务线员工
🔌 连接器腾讯会议 + 乐享 + 企业微信
⚡ PROMPT 示例
把今天上午 10 点的腾讯会议按"客户拜访 7 字段"模板整理会议纪要，写入乐享知识库"客户拜访会议纪要"文件夹下，标题用"客户名_日期"，正文末尾附录制回放链接，写完后把页面链接通过企微发给我和销售总监张三。
📈
跨多场会议的内容总结
03 / 08
日常工作中，周报/月报是高频需求，需汇总多场会议内容；针对同一议题组织的多次讨论会，也常需要做整体性的内容总结与回顾。
👥 适用对象全员（销售、研发、运营、管理者）
🔌 连接器腾讯会议 + 乐享知识库
⚡ PROMPT 示例
读取 5 月 25 日至 5 月 29 日的所有客户拜访相关的腾讯会议录制文件，生成工作周报，最终输出 HTML 单页报告，并在乐享知识库"工作周报存档"位置存档，文件名按"时间+周报总结"命名，完成后把链接发给我。
🚀
销售拜访前准备 + 全流程闭环
04 / 08
拜访前自动调用企查查拉取目标企业基本信息，结合知识库产品资料生成客户谈参；拜访中通过腾讯会议录制；拜访后自动生成纪要、更新 CRM 商机、提取客户提问形成待办派发，并在每周五自动汇总成周报。
👥 适用对象销售、客户经理、渠道运营
🔌 连接器企查查 + 乐享 + 腾讯会议 + 文档 + 企微
⚡ PROMPT 示例
我本周周五下午 5 点要拜访 XXX 集团 CIO，目标推销 XXX 产品：① 用企查查查公司信息、股权、风险；② 从乐享知识库挑制造业大客户产品方案；③ 输出一页纸客户谈参。拜访结束后我会发会议号给你，请按拜访 7 字段模板生成纪要 → 写入腾讯文档商机表 → 客户问题转待办，通过企业微信下发给相关人员。
🛡️
会议合规 / 风险审计
05 / 08
合同评审、商务谈判、对外承诺类会议结束后，WorkBuddy 扫描转写文本，识别"承诺类语句""价格类承诺""合规风险词"，输出风险点清单与原文锚点，自动归档到法务/审计专员邮箱。
👥 适用对象法务、采购、销售运营、风控
🔌 连接器腾讯会议 + QQ 邮箱 + 乐享
⚡ PROMPT 示例
扫描本周所有标注"商务谈判"的腾讯会议录制，识别 ① 口头价格承诺 ② 交付时间承诺 ③ 兜底/保证类话术 ④ 数据/隐私合规风险词，输出风险清单（含原话+时间戳+录制链接），存入乐享"项目合规风险"文件夹，并以 PDF 形式发邮件给法务总监并抄送我。
💼
面试 / 招聘智能评估
06 / 08
HR 通过腾讯会议进行候选人面试，WorkBuddy 基于岗位需求与面试过程，从专业、沟通、匹配度、文化等多维度对候选人进行评分，生成结构化候选人画像，写入招聘系统或腾讯文档候选人表。
👥 适用对象HR、招聘经理、用人部门负责人
🔌 连接器腾讯会议 + 腾讯文档 + 企业微信
⚡ PROMPT 示例
读取今天 14:00 与候选人"王某某-资深架构师岗"的腾讯会议录制，按"分布式系统/团队管理/业务理解/沟通表达"四维评估，每维 1-5 分并附原话证据，生成候选人画像页写入腾讯文档"候选人评估表"，结果通过企业微信同步给招聘负责人 XXX。
🎓
培训 / 内训知识沉淀
07 / 08
内部培训、技术分享、产品宣讲等录制后，自动生成"金句卡片+章节笔记+测验题"三件套，沉淀进乐享知识库形成可复用培训素材包。
👥 适用对象培训发展、培训讲师、企业大学
🔌 连接器腾讯会议 + 乐享知识库
⚡ PROMPT 示例
把昨天 19:00 的"AI Agent 工程化分享"腾讯会议录制处理成培训素材包：① 每 5 分钟一段章节笔记；② 抽 10 句金句做卡片；③ 围绕核心知识点出 8 道测验题（4 单选+4 简答）；全部写入乐享知识库"内训培训会"文件夹下，文件名为"AI Agent 工程化分享"，并在末尾附录制视频。
📖
在线教学课程质量评估
08 / 08
学校/培训机构/企业大学通过腾讯会议开展在线课程，课后自动按"教学内容/讲授清晰度/互动参与/节奏控制/课堂氛围"五维评估课程质量，结合学生发言时长、提问次数、答疑响应时延等客观指标，输出课程质量报告。
👥 适用对象教研负责人、培训主管、授课教师、教务管理
🔌 连接器腾讯会议 + 乐享 + 企业微信
⚡ PROMPT 示例
帮我分析腾讯会议[会议号 xxx]的课程录制，按"教学内容/讲授清晰度/互动参与/节奏控制/课堂氛围"五维评估（满分 100），并附学生发言时长占比、提问次数、答疑平均响应时延等客观数据，最终输出 HTML 课程质量评估报告，保存到乐享"课程质量评估"文件夹下，文件名按"课程名+授课老师+日期"命名，链接通过企业微信发给教研组长和授课老师。
